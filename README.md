# ZARLI (رجالي) - واتساب فقط

موقع تخفيضات رجالي بنمط قريب من صفحات السيل:
- فلترة حسب القسم/المقاس/اللون والسعر
- سلة
- إرسال الطلب مباشرة على واتساب

## التشغيل

الموقع ثابت (Static). شغّله بأي Static Server ثم افتح الصفحة.

مثال بـ Python (لو موجود):

```powershell
python -m http.server 8000
```

ثم افتح:
- http://127.0.0.1:8000
